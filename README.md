# Kiva-Voice-Assistant

Kiva is a voice assistant built in Python that helps you perform various tasks such as:
- Searching the web
- Playing songs
- Fetching the weather
- Reading the latest news
- Opening websites

## Features:
- Speech Recognition
- Text-to-Speech (TTS)
- Command Processing
- OpenAI Integration for unrecognized commands
